# Concurso Público - Lei
## Aplicativo Android

Este aplicativo publicado na play store nesse link [App Concurso Público lei](https://play.google.com/store/apps/details?id=ggtec.lei_concursospublicos).
Com ele e possível acessar as leis federais de forma mais interativa com possibilidades de marcar os textos comentar seus trechos e ouvir atravez da tts(text to speech) do google, o aplicativo também cria historico das leis acessadas e da possibilidade de acessar mais de 24.000 leis.

## Tecnologia utilizada

Aplicativo _Android Nativo_ feito em java utilizando o _material design_ acesso a _api_ própria com o _library volley_ além de faze uso do banco interno _sqlite_ e _tts(text to speech)_ .
